Zerno Aesthetic

Zerno — (from Russian Зерно, meaning Grain).

I've always loved film grain. There's something warm and alive about it that digital photos often lack. Most tools that do this are either bloated, overly complex, or just don't nail the feel I'm after.

So I built my own.

Part of the Visual Dep. of SAM.

Hestikon — 2026

Technical Passport

Honestly, this started as a pure vanity project. Then I stumbled on Dither Me This and realized the same grain effect could actually make images lighter for the web. So it became useful by accident.

Visual Philosophy: An homage to the "Technical Aesthetics" journal - a Soviet design publication that believed beauty and function are the same thing.

Luma-Grain Synthesis: Noise is injected into the luminance channel only, so the grain is monochromatic and doesn't mess with your colors.

Dithering Effect: Dense grain breaks up the flat pixel blocks that JPEG compression loves to mangle. Fewer artifacts, cleaner result.

Smart Optimization: A barely-there softness pass (0.35px) smooths out sensor noise before encoding, so the compressor has an easier job.

AI Ethics & Curation: The technical heavy lifting was done by AI. I decided what it should do and how it should feel.

Privacy Protocol

Zero Tracking: No cookies, no analytics, no tracking pixels.

Offline Capability: Once the page is loaded, it works without internet.
